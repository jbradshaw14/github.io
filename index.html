Lightweight PLC Data Logging

Log S7 PLC data to Excel — without the heavy historian.

Process Journal Pro is a simple Windows app for capturing S7 process values right at the machine. Point it at a non-optimized DB, choose the period, and export clean Excel workbooks automatically. Crash-resilient, operator-friendly, and priced for small teams.

How it works

  1. Create a non-optimized DB with your values (array of REAL/INT/DINT).
  2. Enter PLC connection + offsets.
  3. Pick a save folder. Press Start.
  4. Stop to finalize an Excel workbook (or auto-recover if power is lost).

Process Journal Pro reads with snap7 and writes with openpyxl. It does not modify your PLC program.
